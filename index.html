My applications

Keepr

An application that enables users to share and organize visual content by creating “Keeps” and saving them to themed “Vaults”. Users can browse popular keeps, save them to their collections, and create public or private vaults. The app is built with a Vue.js frontend, Vue-Router, DotNet WebApi for the backend, MySQL database, and utilizes Auth0 for authentication and Dapper for ORM.

Fun Events

An event-planning application that enables users connect for gatherings like concerts, expos, and sporting events. The app features a Vue.js frontend with Vue-Router for navigation, while the backend is powered by an Express and Node.js server with MongoDB and Mongoose handling data storage. Auth0 is integrated for secure user authentication, and a suite of API tests ensures the server meets functional specifications.

The Network

A social application that enables users to connect, share posts, and interact with each other. Users can create, edit, and delete posts, like others’ posts, and manage their profiles. The app includes search functionality to find users and posts and displays random advertisements on each page. Built using Vue.js and Vue-Router for the frontend, the app integrates Auth0 for user authentication and interacts with a backend API for managing user profiles, posts, and ads.

Inspire

A "to-do" list application utilizing multiple asynchronous calls, multiple services and controllers, and user authentication (via Auth0) to demonstrate students’ ability to render complex, real-time data into a cohesive and visually engaging application.

Jot

A feature-rich note-taking application demonstrating MVC architecture, persistent data storage using local storage, dynamic data rendering, and CRUD operations for organizing user notes effectively.

Moon Miner

A space-mining game application demonstrating DOM manipulation, Bootstrap styling, and event handling to simulate resource mining, with upgrades for automatic and manual resource collection.
